## 1. v0.4.0-dev 주요 변경사항

### 핵심: `/api/search` JSON에 `doc_name` 필드 노출

v0.3.0 리포트의 차기 개선 권고(IMP-02 API layer 미연동)를 기준으로, API 응답 shape를 보완했다.

- `internal/api/types.go`
  - `SearchHit`에 `DocName string` 추가
  - Python `retrieve_and_enrich` raw 결과의 `doc_name` 매핑 추가
- `internal/api/search_shape.go`
  - 허용 필드 목록(`allowedSearchResultJSONKeys`)에 `doc_name` 추가
  - `detail=summary`, `detail=standard` preset에 `doc_name` 포함
  - sparse marshal에서 `doc_name` 직렬화 추가
- `internal/api/search_shape_test.go`
  - detail preset 길이/순서 검증 갱신
  - `doc_name` 직렬화 검증 케이스 추가

### 단위 테스트

```bash
GOTOOLCHAIN=auto go test ./internal/api/...
```

결과: `ok   gopedia/internal/api`
