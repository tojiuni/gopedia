## 변경 내용

이 버전은 **Pipeline(Phloem/Xylem) 변경 없음**. `/api/answer` 엔드포인트 및 Answer Agent 레이어 추가.

| 레이어 | 변경 여부 | 내용 |
|--------|---------|------|
| **Phloem** (인제스트/청킹/임베딩) | ❌ 없음 | — |
| **Xylem** (검색/retriever/restorer) | ❌ 없음 | — |
| **API** | ✅ 추가 | `GET /api/answer` 엔드포인트 신규 |
| **Agent** | ✅ 신규 | `flows/xylem_flow/answer_agent.py` — LLM tool-calling 계층형 탐색 |
| **K8s** | ✅ 추가 | `OLLAMA_CHAT_URL`, `OLLAMA_CHAT_MODEL` env 추가 |
