# RAG Test Reports

## 파일 명명 규칙

```
<version>_<date>_<target>.md
```

예: `v0.1.0_2026-04-01_neunexus-gopedia.md`

## 버전 관리

```bash
# 태그 목록
git tag --list

# 태그 생성 (릴리즈 시)
git tag v0.2.0 -m "릴리즈 설명"
git push origin v0.2.0
```
